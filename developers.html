Quick Start

Get up and running in minutes

01

Get API Key

Sign up for a free developer account and get your API key. 100 free API calls per month included. 

# Sign up at:
https://api.exabytesai.com/signup
02

Install SDK

Install the SDK for your preferred language. Python, JavaScript, Unity, Unreal available. 

# Python
pip install exabyte-spatial-ai

# JavaScript
npm install @exabyte/spatial-ai
03

Make First Call

Reconstruct your first 3D scene from an image. Get results in seconds. 

from exabyte import SpatialAI

client = SpatialAI(api_key="YOUR_KEY")
result = client.reconstruct("image.jpg")
print(result.splat_url)

API Reference

Complete REST API documentation

üîÑ Reconstruction API

Convert images and videos to 3D scenes using gaussian splatting and neural rendering. 

POST https://api.exabytesai.com/v1/reconstruct

{
  "media_url": "https://example.com/image.jpg",
  "quality": "high",
  "format": "gaussian_splat",
  "options": {
    "point_cloud": true
  }
}
  • Image reconstruction (single/multi-view)
  • Video reconstruction (temporal consistency)
  • Output formats: splat, mesh, point cloud
  • Processing time: 10s - 5min depending on input

üé® Generation API

Generate navigable 3D environments from text prompts or images. 

POST https://api.exabytesai.com/v1/generate

{
  "prompt": "modern office interior",
  "style": "photorealistic",
  "size": "medium",
  "format": "gaussian_splat"
}
  • Text-to-3D generation
  • Image-to-3D enhancement
  • Style control (photorealistic, artistic)
  • Scene size options (small, medium, large)

Editing API

Modify 3D scenes with AI-powered editing operations. 

POST https://api.exabytesai.com/v1/edit

{
  "scene_id": "scene_abc123",
  "operation": "add_object",
  "params": {
    "object": "chair",
    "position": [0, 0, 0]
  }
}
  • Add/remove objects
  • Change lighting and materials
  • Inpaint and outpaint scenes
  • Style transfer

üìä Management API

Manage scenes, check status, download results. 

GET https://api.exabytesai.com/v1/scenes/:id

{
  "id": "scene_abc123",
  "status": "completed",
  "created_at": "2026-01-01T00:00:00Z",
  "download_url": "https://..."
}
  • List scenes
  • Get scene status
  • Download results
  • Delete scenes
